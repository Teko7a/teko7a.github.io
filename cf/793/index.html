A. Palindromic Indices

题意

给定一个 回文 串 \(s\)，删去一个字符之后其依然回文的删法有多少种？

解答

因为初始回文，只需考虑中心扩展的方法。

展开代码 
void solve() {
  int n;
  std::string s;

  std::cin >> n >> s;

  int mid = n / 2, l = mid - 1;
  while (l >= 0 && s[mid] == s[l]) {
    l -= 1;
  }

  std::cout << (mid - l - 1) * 2 + (n & 1) << '\n';
}

复杂度： \(\mathcal O(n)\)

B. AND Sorting

题意

给定一个 \(0 \sim n - 1\) 的排列，初始时 一定 是无序的。给出下述排序方法，问其中的 \(X\) 最大是多少：

  1. 选择两个数，记为 \(p_i, p_j\) 满足 \(p_i \& p_j = X\)。
  2. 交换他们。

解答

只需要考虑那些未排序好的位置。与运算必然不会得到比原先更大的数，因此 \(X \in [0, n)\)。 将这些位置上的数按位与起来就是答案。

证明也十分简单：考虑 \(p_i \& p_j =X\) 时交换策略：\(\mathrm{swap}(p_i, X)\rightarrow \mathrm{swap}(p_i, p_j) \rightarrow \mathrm{swap}(p_j,X)\)。

展开代码 
void solve() {
  int n, p = INT_MAX;
  std::cin >> n;

  for (int i = 0, x; i < n; ++i) {
    std::cin >> x;
    if (x != i) {
      p &= a[i];
    }
  }

  std::cout << p << '\n';
}

复杂度： \(\mathcal O(n)\)

C. LIS or Reverse LIS?

题意

给定序列 \(\{S\}\)，你可以随意安排其顺序，问最大的 \(\min(\mathrm{LIS}, \mathrm{LIS}_{rev})\) 是多少？

其中 \(\rm LIS\) 指的是 最长严格上升子序列长度，而 \(\mathrm{LIS}_{\_rev}\) 指其反序列。

解答

我们熟知 \(\mathrm {LIS}_{rev} = \mathrm{LDS}\) —— 最长严格下降子序列长度。

本题要求 严格，因而出现次数超过 \(1\) 次的都可当作出现了 \(2\) 次。我们希望更多的数能够同时出现在 \(\rm LIS\) 与 \(\rm LDS\) 中，这些点当然是在原序列中出现 多次 的那些数，记为 \(x\) 个。

剩下的那些仅出现一次的元素只能平均分配到 \(\rm LIS\) 与 \(\rm LDS\) 中了，记为 \(y\) 个。特别注意当 \(y\) 为奇数时，最大值可以同时贡献到两个序列中于是最终贡献为 \(\Big\lceil \dfrac{y}{2} \Big\rceil + x\)。

展开代码 
void solve() {
  int n;
  std::cin >> n;

  std::unordered_map<int, int, custom_hash> mp;

  int cnt = 0;
  for (int i = 0, x; i < n; ++i) {
    std::cin >> x;
    mp[x] += 1;
  }

  for (auto &[k, v] : mp) {
    cnt += v == 1;
  }

  std::cout << ++ cnt / 2 + mp.size() - -- cnt << '\n';
}

复杂度： \(\mathcal O(n)\)

D. Circular Spanning Tree

题意

环上有 \(n\) 个点，顺时针分别为 \(1, 2, \cdots, n\)，给定一个序列 \(s\)。

要求将这 \(n\) 个点连为一棵树，且 \(\delta_i \equiv s_i \pmod 2\)。同时要求 这些边 除端点外不能有交。

\(n \in [2, 2 \times 10 ^ 5]\)

解答

我趣，这题写了好久啊 😭。赛后看各位大神的代码发现自己写的好复杂。

因为 \(n \ge 2\)，所以这棵树一定有超过两个叶子节点，它们的度均为奇数，即 \(\displaystyle \sum_{i = 1} ^ n s_i \gt 0\)。

注意到 \(n = 1\)时则不需要此限制。

另一限制为 \(\displaystyle\sum_{i = 1}^n s_i \equiv 0 \pmod 2\)，这是因为 \(s_i\) 限制的是某节点的度之奇偶，而一棵树的入度与出度显然是一样的。

接下来的构造十分容易，我们只需要将所有的 \(00\cdots001\) 连成 \(\langle\leftarrow 0 - 0 - \cdots - 0 - 0 - 1\rangle\) 随后再将所有的 \(\leftarrow 0\) 连起来就可以了。

展开代码 
void solve() {
  int n;
  std::string s;

  std::cin >> n >> s;

  int o = std::count(s.begin(), s.end(), '1');
  if (!o || o & 1) {
    return std::cout << "NO\n", void();
  }

  std::vector<std::pair<int, int>> ans, chain;

  for (int i = 0; i < n; ++i) {
    if (s[i] == '1') {
      int pos = i;
      while (s[(pos + 1) % n] == '0') {
        int nxt = (pos + 1) % n;
        ans.emplace_back(pos, nxt);
        pos = nxt;
      }
      chain.emplace_back(i, pos);
    }
  }

  for (int i = 1; i < (int) chain.size(); ++i) {
    ans.emplace_back(chain[0].second, chain[i].second);
  }

  std::cout << "YES\n";

  for (auto &[s, e] : ans) {
    std::cout << s + 1 << ' ' << e + 1 << '\n';
  }
}

复杂度： \(\mathcal O(n)\)

void solve() {
  int n, m;
  std::cin >> n >> m;

  std::vector<int> p(n);
  for (auto &i : p) {
    std::cin >> i;
    --i;
  }

  auto g = multivector<int>(n, 0, 0);
  std::map<std::pair<int, int>, int> id;

  for (int i = 0, u, v; i < m; ++i) {
    std::cin >> u >> v;
    --u, --v;
    g[u].push_back(v);
    g[v].push_back(u);
    id[{u, v}] = id[{v, u}] = i + 1;
  }

  std::vector<int> ord(n, -1);

  for (int i = 0; i < n; ++i) {
    if (~ord[i]) {
      continue;
    }
    std::vector<int> a;

    int k = 0;
    for (int y = i; ord[y] == -1; y = p[y]) {
      ord[y] = k++;
      a.push_back(y);
    }
#define MOD(X) (X + k) % k
    for (auto &x : a) {
      std::sort(g[x].begin(), g[x].end(), [&](int i, int j) {
        return MOD(ord[i] - ord[x]) < MOD(ord[j] - ord[x]);
      });
    }
  }

  std::vector<int> cur(n);

  auto check = y_combinator([&](auto check, int x) -> void {
    if (cur[x] == (int)g[x].size()) {
      return;
    }
    int y = g[x][cur[x]];
    if (g[y][cur[y]] != x) {
      return;
    }
    std::cout << id[{x, y}] << " ";
    cur[x] += 1, cur[y] += 1;
    check(x), check(y);
  });

  for (int i = 0; i < n; ++i) {
    check(i);
  }
}

复杂度： \(\mathcal O(<++>)\)
