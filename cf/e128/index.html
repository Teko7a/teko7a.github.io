A. Minimums and Maximums

题意

一个包含 l1∼r1l_1 \sim r_1l1​∼r1​ 个最小值，l2∼r2l_2 \sim r_2l2​∼r2​ 个最大值的数组最短是多少？

解答

令 l1≤l2l_1 \le l_2l1​≤l2​，若 l2∈[l1,r1]l_2 \in [l_1, r_1]l2​∈[l1​,r1​]，则可以全设置为一种数值，数量为 l2l_2l2​。否则为 l1+l2l_1 + l_2l1​+l2​ 个数。

void solve() {
  int a, b, c, d;
  std::cin >> a >> b >> c >> d;

  if (a > c) {
    std::tie(a, b, c, d) = std::tuple{c, d, a, b};
  }
  
  std::cout << c + a * (c != std::clamp(c, a, b)) << '\n';
}

B. Robots

题意

二维地图上有一些棋子，他们的移动步伐一致，问是否可能在任何棋子都不越界情况下将某个棋子移动至 (0,0)(0, 0)(0,0)。

解答

想象成把终点移动至某个棋子。

要求即终点的横、纵两维同时切割到一个棋子。

展开代码 
void solve() {
  static constexpr int inf = 0X3F3F3F3F;
  int n, m, x = inf, y = inf;
  std::cin >> n >> m;
 
  auto grids = multivector<char>(n, m, 0);
  for (int i = 0; i < n; ++i) {
    for (int j = 0; j < m; ++j) {
      std::cin >> grids[i][j];
      if (grids[i][j] == 'R') {
        x = std::min(x, i);
        y = std::min(y, j);
      }
    }
  }
 
  std::cout << (grids[x][y] == 'R' ? "YES\n" : "NO\n");
}

复杂度： O(n×m)\mathcal O(n \times m)O(n×m)。

由于题目保证至少有一个棋子，因而访问到的空间都是合理的。

C. Binary String

题意

希望 max⁡{\max\{max{ 剩下的 000，删去的 111 }\}} 尽可能 小。

为此，你可以从两端删去任意多的字符，求此值。

解答

从开头和结尾删去一些字符，实际上就是找到一个子区间。

双指针扫描，在 cnt0<cnt1cnt_0 \lt cnt_1cnt0​<cnt1​ 情况下一直右移右端点。右端点右移时，000 可能变多，（从右侧）删去的 111 可能减少；而左端点右移时，删去的 111 可能增多，留下的 000 可能减少。两者至多也只会各自遍历一次数组，因而可在线性时间解决。

展开代码 
void solve() {
  std::string s;
  std::cin >> s;

  int n = s.size();
  int o = std::count(s.begin(), s.end(), '1');
  int a = o;

  for (int i = 0, j = -1, z = 0; i < n; ++i) {
    while (j < n - 1 && z < o) {
      j += 1;
      z += s[j] == '0';
      o -= s[j] != '0';
    }
    a = std::min(a, std::max(z, o));
    z -= s[i] == '0';
    o += s[i] != '0';
  }

  std::cout << a << '\n';
}

复杂度： O(n)\mathcal O(n)O(n)
