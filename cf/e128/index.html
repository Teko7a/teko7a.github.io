A. Minimums and Maximums

题意

一个包含 \(l_1 \sim r_1\) 个最小值，\(l_2 \sim r_2\) 个最大值的数组最短是多少？

解答

令 \(l_1 \le l_2\)，若 \(l_2 \in [l_1, r_1]\)，则可以全设置为一种数值，数量为 \(l_2\)。否则为 \(l_1 + l_2\) 个数。

展开代码 
void solve() {
  int a, b, c, d;
  std::cin >> a >> b >> c >> d;

  if (a > c) {
    std::tie(a, b, c, d) = std::tuple{c, d, a, b};
  }
  
  std::cout << c + a * (c != std::clamp(c, a, b)) << '\n';
}

复杂度： \(\mathcal O(1)\)

B. Robots

题意

二维地图上有一些棋子，他们的移动步伐一致，问是否可能在任何棋子都不越界情况下将某个棋子移动至 \((0, 0)\)。

解答

想象成把终点移动至某个棋子。

要求即终点的横、纵两维同时切割到一个棋子。

展开代码 
void solve() {
  static constexpr int inf = 0X3F3F3F3F;
  int n, m, x = inf, y = inf;
  std::cin >> n >> m;
 
  auto grids = multivector<char>(n, m, 0);
  for (int i = 0; i < n; ++i) {
    for (int j = 0; j < m; ++j) {
      std::cin >> grids[i][j];
      if (grids[i][j] == 'R') {
        x = std::min(x, i);
        y = std::min(y, j);
      }
    }
  }
 
  std::cout << (grids[x][y] == 'R' ? "YES\n" : "NO\n");
}

复杂度： \(\mathcal O(n \times m)\)。

由于题目保证至少有一个棋子，因而访问到的空间都是合理的。

C. Binary String

题意

希望 \(\max\{\) 剩下的 \(0\)，删去的 \(1\) \(\}\) 尽可能 小。

为此，你可以从两端删去任意多的字符，求此值。

解答

从开头和结尾删去一些字符，实际上就是找到一个子区间。若固定区间的任一端点，另一端的变化是单调的。

双指针枚举，在 \(cnt_0 \lt cnt_1\) 情况（保证单调性）下一直右移右端点。右端点右移时，\(0\) 可能变多，（从右侧）删去的 \(1\) 可能减少；而左端点右移时，（从左侧）删去的 \(1\) 可能增多，留下的 \(0\) 可能减少。

展开代码 
void solve() {
  std::string s;
  std::cin >> s;

  int n = s.size();
  int o = std::count(s.begin(), s.end(), '1');
  int a = o;

  for (int i = 0, j = -1, z = 0; i < n; ++i) {
    while (j < n - 1 && z < o) {
      j += 1;
      z += s[j] == '0';
      o -= s[j] != '0';
    }
    a = std::min(a, std::max(z, o));
    z -= s[i] == '0';
    o += s[i] != '0';
  }

  std::cout << a << '\n';
}

复杂度： \(\mathcal O(n)\)

两者至多也只会各自遍历一次数组，因而为线性时间。
