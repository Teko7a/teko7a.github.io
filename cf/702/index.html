CF702

A. Dense Array

题意

希望对于任两相邻元素都有：，在给定的数组中 插入 最少的数以满足上述条件。

解答

数据量非常小，直接模拟即可。

展开参考代码 
void solve() {
  int n;
  std::cin >> n;

  std::vector<int> a(n);
  for (auto &i : a) {
    std::cin >> i;
  }

  int cnt = 0;
  for (int i = 0; i < n - 1; ++i) {
    int M = std::max(a[i], a[i + 1]);
    int m = std::min(a[i], a[i + 1]);
    while (M > 2 * m) {
      m *= 2;
      cnt += 1;
    }
  }
 
  std::cout << cnt << '\n';
}
